# brutus-pens-11ty

```zsh
# build files to prepare for deploy
yarn build

# yarn gulp + yarn 11ty
yarn dev

# remove build files, reset /dist folder
yarn cleanup

# watch and compile 11ty files
yarn 11ty

# watch and compile SCSS files
yarn gulp

# compile SCSS files (once)
yarn scss

# watch and compile JS and Svelte files
yarn rollup
```
